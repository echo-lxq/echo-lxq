# HTML #

### 1.flex：1 ###

即为flex-grow：1，经常用作自适应布局，将父容器的display：flex，侧边栏大小固定后，将内容区flex：1，内容区则会自动放大占满剩余空间。

### 3.行内标签与块级标签 ###

	{/* 1.行级标签 */}

            {/* 1.1特点 */}
            {/* 可以和其他元素保持在同一行,不可以自动换行,但不能设置宽高 */}

            {/* 1.2常见行级标签 */}
            <a></a>
            <span></span>
            <u></u> {/* 下划线 */}
            <em></em> {/* 强调 */}
            <i></i> {/* 斜体 */}
            <sub></sub> {/* 下标 */}
            <sup></sup> {/* 上标 */}

    {/* 2.块级标签 */}

            {/* 2.1特点 */}
            {/* 不可以和其他元素保持在同一行(独占一行)，可以自动换行，能设置宽高 */}

            {/* 2.2常见块级标签 */}
            <div></div>
            <p></p>
            <h1-h6></h1-h6>
            <ul></ul>
            <li></li>
            <dl></dl> {/* 定义列表，跟ul...li类似 */}
            <dt></dt> {/* 定义列表中的项目 */}
            
    {/* 3.行内块级标签 */}

            {/* 3.1特点 */}
            {/* 可以和其他元素保持在一行，还能设置宽高 */}

            {/* 3.2常见行内块级标签 */}
            <textarea name="" id="" cols="30" rows="10"></textarea>
            <input type="text" />
            <img src="" alt="" />
            <button></button>
